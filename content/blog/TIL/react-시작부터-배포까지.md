##react 왜 사용하나요?
###SPA
spa는 rendering에 필요한 JS, CSS, HTML 파일을  
한꺼번에 받아온 후, server에는 data만 요청한다(AJAX).  

###REACT vs VUE
react나 vue나 기술적인 차이는 별로 없음
react 연봉이 더 높다

###React의 특징

- virtual DOM 사용, rendering 빠르다
- component 높은 재사용성, 빠른 개발
- JSX 선언적, 직관적, 높은 생산성
